# 📊 codo1 AI Business Analysis System

نظام ذكي لتحليل الأنشطة التجارية بناءً على أسئلة مخصصة وعرض واجهة مستخدم (Dashboard) تفاعلية ومؤتمتة بالكامل، مع تكامل مع بوت تيليجرام لتغيير حالة الطلبات مباشرة.

تمت تهيئة المشروع بالكامل ليعمل بشكل هجين (محلياً وفي نفس الوقت جاهز للرفع على **Vercel** كـ Serverless).

---

## 🚀 طريقة الرفع والتشغيل على Vercel

يدعم المشروع قواعد بيانات **Vercel KV (Upstash Redis)** بشكل تلقائي بدون الحاجة لأي تعديل في الكود.

### الخطوة 1: الرفع إلى GitHub
قم برفع مجلد المشروع بالكامل إلى مستودع (Repository) خاص بك على GitHub.

### الخطوة 2: إنشاء المشروع على Vercel
1. اذهب إلى موقع [Vercel](https://vercel.com) وقم بتسجيل الدخول.
2. اضغط على **Add New -> Project** ثم اختر مستودع المشروع من GitHub.
3. في صفحة الإعدادات، **لا تقم بتغيير أي مسارات أو إعدادات افتراضية** (سيقوم Vercel بقراءة ملف `vercel.json` تلقائياً لتشغيل الباك إند كـ Serverless والفرونت إند كـ Static Assets).

### الخطوة 3: إضافة قاعدة البيانات (Vercel KV)
1. بعد تهيئة المشروع على Vercel، اذهب إلى تبويب **Storage** في لوحة تحكم المشروع على Vercel.
2. اختر **KV** (قاعدة بيانات Redis سريعة ومجانية تقدمها فيرسيل).
3. اضغط على **Create New** ثم اختر ربطها بمشروعك الحالي (**Connect**).
4. سيقوم Vercel تلقائياً بحقن متغيرات البيئة المطلوبة (`KV_REST_API_URL`, `KV_REST_API_TOKEN`) في مشروعك.

### الخطوة 4: ربط بوت تيليجرام (Telegram Bot)
قم بإضافة متغيرات البيئة التالية في إعدادات المشروع على Vercel (**Settings -> Environment Variables**):
- `TELEGRAM_BOT_TOKEN`: توكن البوت الخاص بك من BotFather.
- `TELEGRAM_CHAT_ID`: رقم الـ Chat ID الخاص بك أو بقناتك لاستقبال الطلبات.

اضغط على **Deploy** لإعادة بناء المشروع وتشغيله. بمجرد إرسال أول طلب، سيقوم النظام تلقائياً بربط الـ Webhook الخاص بتيليجرام ليعمل البوت والردود التفاعلية بشكل فوري!

## 💻 التشغيل المحلي للتجربة (Local Dev)

عند تشغيل المشروع محلياً، يقوم النظام تلقائياً بالتحول لاستخدام ملف **`orders.json`** كقاعدة بيانات محلية، ويقوم بتشغيل خيط فرعي لمراقبة رسائل بوت تيليجرام (Polling).

### المتطلبات
تثبيت المكتبات البرمجية المطلوبة:
```bash
pip install -r requirements.txt
```

### الإعداد
1. قم بفتح ملف `config.json` في المجلد الرئيسي واكتب توكن البوت والـ Chat ID الخاص بك.
2. شغّل الخادم المحلي:
   ```bash
   python app.py
   ```
3. افتح المتصفح على: `http://127.0.0.1:8000`

## 🛠️ محاكي تيليجرام المدمج (للاختبار السهل)
إذا لم يكن لديك بوت تيليجرام جاهز، يمكنك اختبار حالة الطلبات (تأكيد، رفض، تعليق) مباشرة من داخل لوحة التحكم (الداش بورد) عن طريق **لوحة محاكاة تيليجرام** التي قمنا بتطويرها أسفل تفاصيل النشاط في القائمة الجانبية.
